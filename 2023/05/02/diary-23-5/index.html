5月2日

  • 昨日带csm、金宝参观华子，徒步把校园转了个遍。途中去紫荆书咖小坐并各点了杯咖啡，我点的咖啡名为「绝绝紫」，十分滑稽，但味道却不错。这也是我自己第一次去紫荆书咖，感觉氛围、出品、价位都尚可，以后可以经常光顾。晚饭和ty、wg和杰哥一起去吃了涮肉。虽然在北京待了这么久，这却也是我第二次吃涮肉。晚上我们三个提前走了赶往景山，快马加鞭总算抢在闭园前进了公园，但因为金肚子不舒服耽搁了一会时间，在即将到达山顶之时被赶了下来，功亏一篑。不过夜里走在黑灯瞎火的山路上，也别具一番风味，令人想到丽奈与久美子的登山。从公园出来后到附近故宫角楼的对岸闲坐。夜里的故宫十分幽雅，护城河岸边的清风也十分舒服。即便已经九点了，还有许多钓鱼佬备置了鱼竿、探照灯等设备在旁边钓鱼，但似乎只钓得上水草。听闻wg等一行人在清河万象汇吃鲜芋仙，决定立刻赶往相会，不料到达时商城已经即将关门歇业，然而我们已经点好了鲜芋仙，只好托他们打包带出来，但汇合后才发现三份甜品只有一个勺子，待我和csm赶回时冰淇淋都已经化了。在商场门口，ty让我们每人帮他抽了五十发明日方舟的卡，几个人又一起合了影，便只好悻悻而归，本来还想再一起喝点酒却作罢。
  • 今日早上美美地睡到十点才醒。本来csm要去找杰哥一起去人大，却因为杰哥改去北大而csm没约上而作罢。于是我们三个人又一起吃了饭。到之前只吃过外卖的叙利亚餐厅吃了饭，排队等了好一会。价格十分实惠，餐品十分特别，红茶、小吃、茄子蔬菜酱令人印象深刻。吃饭时发现张妈、ozq两人也正在附近吃饭，于是与他们也见了一面。吃完饭时时间已不早，又急匆匆地赶往景山。到了公园附近后待我买好自己的票后另外两人却发现预约已满。大家十分慌张，以为世界线又要收束了。抱着碰碰运气看能不能买现场票的侥幸心理，我们姑且朝着景山走去，此时金却意外地发现又有预约余额，但在他预约之后csm又无法预约了，于是我们猜测有人退票就会放出余额。于是我们边走边刷着手机，祈祷着还有人退票，但许久都不再有余额放出，内心十分焦急，金宝还要赶高铁，如果一时抢不到票那么将来不及上山。直到我们走到公园门口，才终于抢到，长舒了一口气。此时发现现场票已经无法购买，我们还算幸运。帮金拖着行李箱登上了景山。公园里人虽多，但还不至于拥挤。山顶的景致确实不错，近能俯瞰北海、故宫，远能看到北京CBD，可惜白天来还是差点意思，更可惜的是因为金宝要赶高铁我们待了几分钟就只好下山，于是我们约好待他们下次前来北京时再来景山看夜景。送走金宝后csm无事可做，于是我陪他前往国家大剧院参观。途中路过中南海旁边，还真是戒备森严。到国家大剧院旁边时发现交通管制异常严格，似乎是因为人民大会堂在搞活动，没法看到大馒头浮在水面上的景观。于是，我们进入大剧院内部参观。这也是我第一次白天到大剧院参观，这时才发现入场的那段路的玻璃天花板上就是水面，也才领会到大剧院咖啡厅的价格多么离谱。从大剧院出来后想找个能喝东西、充电的地方，于是找到前门大栅栏一处不显眼的小咖啡厅，装潢十分有情调，似乎是由几个漂亮姐姐合伙开的。喝完咖啡，csm便要离京。像这几天这样能与好友相见，一同游玩的机会真是不可多得。在华子待了这么久，却仍没有算得上知交的朋友，也真是有点可悲。珍惜眼下的朋友吧。
  • 检查了一下网络学堂，发现线代的ddl其实是在下周，不知道是我看错了还是延期了，总之可以姑且放开玩了。
  • 近两日校园里游客的数量远超想象，局面之混乱惨不忍睹，种种低素质的现象触目惊心，希望华子能管一管。

5月6日

  • 终于得闲来写日记了。
  • 送走金宝和csm的那个晚上肠胃忽然非常不舒服，以致一夜都没睡好，甚至怀疑自己第二天还能不能去大同。吃了许多囤积的药，收效都甚微。到了早上再次疼醒时，决定外卖订两盒蒙脱石散，吃了两包。所幸此后感觉舒服了一些，没有因此耽搁大同之行。然而，彻底好透还是花了两天时间。肠胃不适加上上火，导致在大同玩的几天都没能好好品尝美食。
  • 抵达大同的时候已近下午了，于是在酒店吃午饭，价格在几天的行程里算是高的，但出品还算不错。酒店的服务员见我们点了羊肉却没有点素糕，还送了我们一块。后便决定步行在古城转一转。一个下午，我们先后寻访了九龙壁、华严寺、善化寺等景点，这些景点古朴雅正，没让人失望。值得一提的是，古城里的鸟非常多，它们盘旋在古建筑的上面，更为古建筑多添一分神韵。此外，途中我还被文庙、关帝庙、帝君庙等小众古建吸引了，于是暂时让爸妈先前往华严寺，而自己则穿梭在古城白墙灰瓦的大街小巷里。到大同的第一天时五一假期尚未结束，古城里人流熙熙攘攘，关帝庙、华严寺附近还进行了公开的演出，好不热闹。虽然古城的很多建筑实际上是现代复建的，但复原的效果可以假乱真，毕竟还是能使人一览其历史风貌。晚饭时还是没有什么胃口，决定去吃刀削面，山西的面条果然十分劲道，价格也不过十块出头，较为实惠。虽然腿已经发酸，但饭后还是决定登临古城墙漫步。说实话，城墙上的夜景并没有达到我的语气，古城里的灯光设置的不好，有些地方亮得至于光污染而有的地方却黑漆漆一片，还能看到许多正在建设的工地和烂尾楼。
  • 第二天，我们包了辆车，计划玩完悬空寺和云冈石窟。本来老妈还想去应县木塔，但时间恐怕不允许了。两处景点都巧夺天工、震撼人心，尤以云冈石窟为甚。可惜为保护文物石窟内部不允许拍照。午饭在凯鸽酒楼吃，美味又实惠，不愧于其口碑。其中沙棘汁只要36一扎，远比其他地方便宜。晚饭则是在好评度也很高的所谓紫泥369粗粮季吃的。餐馆取这样一个名字还真是不同寻常。这顿的用餐体验不如中午，一方面是价格更为昂贵，如沙棘汁竟要88一扎，另一方面则是一些本地特色的菜并不是很合意。
  • 包车的司机对我早有耳闻的耿市长评价很高，一路上给我们讲了许多他的事迹，包括修整古城、建设道路、对付钉子户等。由此可见这是一个极有手腕和行动力、恪尽职守的人民公仆，而他也确实赢得了大同人民的尊敬。
  • 本来听闻大同博物馆值得一游，但我实在太累了想睡懒觉，时间上可能也来不及，于是就干脆放弃了。不过由于晚上睡得太晚，实际上还是没能睡多久（
  • 这次爸妈把新相机带来了，尽管还不太会用，还是迫不及待地带上了。相机的性能果然还是比手机好得多，但不得不说定焦的镜头有些不方便。
  • 本想着五一假期这么长，能够多睡点觉，好好休息休息，但最后安排得过于充实，到处去玩，反而不仅睡得更少，身体更疲劳，ddl也没时间赶了，更没时间进行打游戏、看书、弹琴等娱乐活动（
  • 昨天ty忽然说请我和张妈去北大玩，虽然担心ddl可能赶不完，还是欣然答应。见面才知道原来是他保研面试结束，闲下来了。北大虽然楼宇不如华子宏伟大气，有些也有些破旧了，但校园仿佛公园一般，整体上给人的感觉十分干净、秀气、舒适。此行令人想起了高二参加北大金秋营的时光，当时也游览了北大校园，不过没有逛得像今天这样细致。此次来，还识别出了当年举办活动的理科教学楼，令人感慨万千。要是我没来华子而是去了北大，那么现在当是如何情形呢？晚饭在北大农园三层吃的，中规中矩，和华子校内的点餐餐馆别无二致。倒是晚饭后在一楼吃到的冰淇淋十分惊艳。接着一同去喝了杯喜茶，感觉味道还是不尽人意。最后因方向不同与张妈分别后，ty陪我散步至西门附近，一路聊天畅谈。ty似乎十分想要认识学弟学妹，笑。
  • 今天就是假期的最后一天了，明天还有早八，悲。

5月7日

  • 开学第一天早八课后就昏睡一小时，难绷。
  • 读PaLM论文遇到一大堆不理解的技术细节，看来要学的还有很多啊……
  • 愈发感觉自己对计算机的兴趣实际上不是很坚定，但也不知道自己到底真正热爱什么，现在再来作什么改变代价也很高了。
  • 还是没有退大物，背水一战吧。
  • 晚上和执信约饭群友们一起去北大附近吃潮汕火锅，骑车花了不少时间，但味道、性价比都很不错。此外，这家店有许多别处不常有的食材，还有免费自取的西瓜。在大同的时候凯鸽也送了西瓜。印象中小时候许多饭馆都会送水果，现在似乎不常见了。吃完饭还步行一起去新中关买了茶话弄。中关村这块虽然很近，却直到这两天之前都没来过。
  • 今天是勃拉姆斯诞辰一百九十周年，他算是我最喜欢的音乐家之一了。今天也是csm诞辰二十周年（雾），然而我忘了，罪过。不过我是向来不记别人生日的，所以将心比心如果有人记得我的生日我会很感动。

5月9日

  • 今天去大剧院听了贝奏，本来事情比预想的多很多不想去了，但毕竟钱已经花了……
  • 好久没现场听钢琴独奏了，虽然位置有点偏，但感受还是远胜录音，也更感慨自己弹的都是什么答辩。
  • 月光三能明显听出一些错音……不过加演了暴风雨三，血赚！
  • 结束的时候被我妈以前的同学认出来了，她没有见过我，只看过我照片；也不住北京，只是这两天来看她女儿，神奇。。
  • 学长似乎有些高估我们的能力，最后一个入门任务未免太难了。

5月10日

  • 学习状态在五一假期后还是没有改观。
  • 买的显示屏不是很满意，但完全符合我要求的要贵得多，也懒得退货了，就先这样吧。
  • 花了大半天来做Instruction Tuning的入门任务，困难实在太大了，做不下去了……感觉学长在布置的时候有点欠考虑。明天问问他是怎么回事吧。虽然不是毫无收获，但投入这么多时间，赶ddl又更加困难了。

5月12日

  • 这几天为了赶ddl忙死了，昨天晚上肝作业到两点半才睡，今天上课屡屡昏睡。这个周末事情也不少，不过今晚还是姑且放松了一下。
  • Instruction Tuning的入门任务果然是哪里搞错了……学长似乎打算这就把实际的工作布置给我们了，然而我至今尚不清楚组里在做的项目具体是什么（
  • 日记越写越流水账了，不符合当初的设想，不过也没办法，近来实在没什么时间。一些沉淀的思想，待闲下来了再记录吧。

5月14日

  • 本来想中午去莆田吃蛏子，结果找不到人一起去。读大学快两年了，认识的人不少，朋友却寥寥，可叹。

5月16日

  • 终于约到wg和ty一起去吃莆田了，味道没让我失望。
  • 搞线性回归分析搞了一天，这门课的东西实在太多太杂了。

5月17日

  • 有点低估近来ddl的量了。
  • 十分后悔报名了班委，不但没能拉近和同学们的关系，还被丢了一堆杂七杂八的活。
  • 居然又忘记了交作业，又是隔了块一周才发现，崩溃。

5月20日

  • 昨天二队沙龙音乐会，也是学员班首演。并没有什么实感。吹到后面都不知道自己在吹什么了。上大学两年以来第一次把西装穿出去。
  • 首演最后拍合照的时候更感受到自己的孤独。本想加入乐队能多认识一些人，结果还是没交到什么朋友。到聚餐的时候基本都是按声部坐的，圆号没来什么人，现场也没几个我的熟人，十分尴尬。此外，考虑到时间不早了，再晚回去就没法洗澡，且穿白衬衫吃火锅有点危险，于是到偷偷溜走，在楼下打包了一份KFC作夜宵就回去了。
  • 今天下午排练难得是曹指来排，久违地感受到了排练的快乐。
  • 又有换宿舍的机会了，还是很纠结。同时也纠结要不要加sfy的支队暑假去实践。

5月21日

  • 被胥导约饭后最终还是拒绝了换宿舍。如果要换的话，必须两周内搬完，有点劝退。而且据胥导所说他们那届年级前十有三个转专业过来的，都没换宿舍。
  • 下午通识课组织去长庚医院参观。医生们十分有趣、可爱，学识也十分渊博，待我们见识了小型化的超声、用于动捕以测量运动能力的XBOX体感摄像头、用于手术的AR设备等前沿技术，也见识了各式各样的结石。现场还给一位很胖的同学用超声查出了脂肪肝，附赠报告一篇（

5月22日

  • 卡里莫名被打了400块勤工补贴，不会是科研补贴吧，感觉自己都没做什么（

5月27日

  • 不知不觉好多天没更了，忙的时候确实会丧失对写日记的兴趣。
  • 最近和hjc关系越来越好了，实在意料之外。
  • 给人智导四子棋debug整得想死，各种报Segment Fault，两次搞到凌晨两点。
  • 科研那边要我上手跑通评测用的脚本，结果到处都是坑，难绷。
  • 今早宿舍进了一只麻雀，它花了好长时间才找到窗口出去，还在zh桌上进行了排遗（
  • 昨天去听了一队演出，学长学姐们还是很厉害。

5月28日

  • zzl上午来华子搞什么体测比赛，于是中午和她还有wg一起去玉树一吃宜宾美食周推出的宜宾菜，感觉味道还可以，但不及预期。
  • 下午和xyc、wg、zhc、lf一起去录给高三学生加油的视频，因为lf要打电话耽搁了不少时间，晚上合排差点迟到。高考又要到了，彼时的我是什么心境，已经记不清了。
  • 晚上是学员班最后一次合排。pxt没到的时候我吹solo简直是灾难，一个人吹和两个人一起吹的感受还是很不一样。中间指导大镲的时候让小号的同学试了一下，感觉耳朵要震聋了。最后一次全曲录音录像的时候感觉正常发挥了，算是给这一年画上了完满的句号。最后一起分了蛋糕吃，好像是什么冰淇淋蛋糕，十分美味。不知不觉就要升入二队了，希望接下来一年能交到更多朋友，继续享受音乐的魅力。

5月31日

  • 人智导四子棋是真的卷不动了，在平台上跑比在本地跑慢太多了。要继续提速就得重构很多代码，试了一下不是很成功。不想再花时间慢慢调试了，边际效益太低，算了。
  • 前天想办法面试音辅，费了好大力气菜联系上，结果与往年不同，今年居然有竞争了，感觉十分危险。此外感觉谢鹏这个人有点傲慢，有点屑（贬义），心里有点不安。
  • 期末实在太忙了，日记更新频率和质量都要下降咯。
